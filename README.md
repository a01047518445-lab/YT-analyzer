# YT-analyzer
조회수·VPH·영향력 지수부터 AI 트렌드 인사이트까지, 유튜브 콘텐츠 성과를 숫자로 보여주는 분석 도구

이 도구는 유튜브 검색 결과를 기반으로
조회수, 구독자 수, 영향력 지수, VPH, 영상 길이, 업로드 시기를 자동 수집·분석합니다.

검색 결과는 기간별·영상 길이별로 필터링할 수 있으며,
연관 키워드 추천과 AI 트렌드 분석을 통해 콘텐츠 기획에 바로 활용할 수 있습니다.

분석된 데이터는 엑셀(CSV) 파일로 저장 가능하며,
영상 썸네일을 즉시 다운로드할 수 있습니다.

입력 언어에 따라 국가 제한 없이 글로벌 유튜브 콘텐츠 분석을 지원합니다.
